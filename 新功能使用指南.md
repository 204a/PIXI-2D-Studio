# 新功能使用指南

## 🎉 三大新系统（2026-04-01更新）

### 1. 复制/粘贴系统

**快捷键:**
- `Ctrl+C` - 复制选中对象
- `Ctrl+V` - 粘贴对象（自动偏移20px）
- `Ctrl+X` - 剪切对象
- `Ctrl+D` - 快速复制（原地复制）

**功能特点:**
- 完整复制对象及其行为事件
- 自动偏移位置，避免重叠
- 状态栏实时提示操作结果

---

### 2. 画布缩放和平移

**缩放操作:**
- `滚轮` - 以鼠标为中心缩放（0.1x ~ 4x）
- 工具栏 `+` 按钮 - 放大
- 工具栏 `−` 按钮 - 缩小
- 工具栏 `重置` 按钮 - 恢复100%

**平移操作:**
- `空格 + 拖拽` - 平移画布
- 空格按下时光标变为抓手

**特色功能:**
- 缩放中心智能定位（鼠标位置）
- 实时显示缩放比例
- 适应内容（右键菜单）

---

### 3. 右键菜单

**对象菜单**（右键对象时）:
- 复制/剪切/粘贴/快速复制
- 删除对象
- 置于顶层/底层
- 锁定/解锁对象
- 显示/隐藏对象
- 清空容器（容器类型）

**画布菜单**（右键空白处）:
- 粘贴
- 显示/隐藏网格
- 重置视图
- 适应内容

**特点:**
- 快捷键提示
- 灰显不可用选项
- 响应当前状态

---

## 📝 使用建议

### 高效工作流

1. **快速复制对象**
   ```
   选中对象 → Ctrl+D → 拖动到新位置
   ```

2. **批量创建**
   ```
   创建第一个 → Ctrl+C → 多次Ctrl+V → 调整位置
   ```

3. **大场景编辑**
   ```
   滚轮缩小查看全局 → 空格拖动定位 → 滚轮放大编辑细节
   ```

4. **快速调整层级**
   ```
   右键对象 → 置于顶层/底层
   ```

### 注意事项

- 复制粘贴会同时复制所有行为事件
- 缩放不影响对象实际大小（仅改变视图）
- 锁定的对象无法选中和编辑
- 隐藏的对象仍会在运行时显示
