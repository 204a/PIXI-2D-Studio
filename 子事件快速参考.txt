├────────────────────────────────────────────────────────────┤

│                                                              │

└────────────────────────────────────────────────────────────┘

┌────────────────────────────────────────────────────────────┐
│ 核心特性                                                     │
├────────────────────────────────────────────────────────────┤
│ ✓ 最多2层嵌套（主 → 子 → 孙）                               │
│ ✓ 独立的条件和动作                                          │
│ ✓ 级联删除（删除父事件时自动删除子事件）                     │
│ ✓ 拖拽排序                                                   │
│ ✓ 支持撤销/重做                                              │
└────────────────────────────────────────────────────────────┘

┌────────────────────────────────────────────────────────────┐
│ 实用模式                                                     │
├────────────────────────────────────────────────────────────┤
│ 🎯 条件分支:                                                 │
│    主: 按键按住                                              │
│      子1: X < 400 → 动作A                                    │
│      子2: X >= 400 → 动作B                                   │
│                                                              │
│ 🎯 连续触发:                                                 │
│    主: 碰撞检测                                              │
│      子: 透明度 < 0.1 → 隐藏                                 │
│                                                              │
│ 🎯 组合条件:                                                 │
│    主: (无条件)                                              │
│      子1: 条件A → 动作A                                      │
│      子2: 条件B → 动作B                                      │
└────────────────────────────────────────────────────────────┘

┌────────────────────────────────────────────────────────────┐
│ 快捷键                                                       │
├────────────────────────────────────────────────────────────┤
│ Ctrl + Z     撤销                                            │
│ Ctrl+Shift+Z 重做                                            │
│ Delete       删除选中对象                                    │
│ G            显示/隐藏网格                                    │
└────────────────────────────────────────────────────────────┘

┌────────────────────────────────────────────────────────────┐
│ 技术细节                                                     │
