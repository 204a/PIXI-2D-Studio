# 对齐功能使用指南

## 🎯 智能对齐系统（2026-04-08新增）

### 功能概述

对齐系统提供专业级的对象布局工具，包括：
- **6种对齐方式**（左/右/上/下/水平居中/垂直居中）
- **2种分布方式**（水平分布/垂直分布）
- **3种大小匹配**（匹配宽度/高度/大小）
- **智能吸附**（拖动时自动对齐到其他对象，带辅助线提示）

---

## 📐 对齐工具栏

### 激活方式

1. **自动显示**：选中 **2个或以上** 对象时，对齐工具栏自动显示在画布左上角
2. **快捷键**：按 `A` 键手动显示/隐藏对齐面板

### 工具栏布局

```
对齐工具
⬅️  ↔️  ➡️    （左对齐 / 水平居中 / 右对齐）
⬆️  ↕️  ⬇️    （顶部对齐 / 垂直居中 / 底部对齐）

分布
⬌ 水平  ⬍ 垂直

匹配大小
宽  高  全
```

## 🎨 使用方法

### 1. 对齐功能（需要至少2个对象）

#### 左对齐
- **操作**：点击对齐工具栏的 `⬅️` 按钮
- **效果**：所有选中对象的左边缘对齐到最左边对象的位置
- **右键菜单**：右键选中对象 → `左对齐`

#### 右对齐
- **操作**：点击 `➡️` 按钮
- **效果**：所有对象的右边缘对齐到最右边对象

#### 顶部对齐
- **操作**：点击 `⬆️` 按钮
- **效果**：所有对象的顶部边缘对齐到最上方对象

#### 底部对齐
- **操作**：点击 `⬇️` 按钮
- **效果**：所有对象的底部边缘对齐到最下方对象

#### 水平居中对齐
- **操作**：点击 `↔️` 按钮
- **效果**：所有对象的水平中心点对齐（计算平均中心位置）

#### 垂直居中对齐
- **操作**：点击 `↕️` 按钮
- **效果**：所有对象的垂直中心点对齐

---

### 2. 分布功能（需要至少3个对象）

#### 水平分布
- **操作**：点击 `⬌ 水平` 按钮
- **效果**：在最左和最右对象之间，均匀分布所有对象
- **计算方式**：自动计算对象间的最佳间距

#### 垂直分布
- **操作**：点击 `⬍ 垂直` 按钮
- **效果**：在最上和最下对象之间，均匀分布所有对象

---

### 3. 匹配大小功能（需要至少2个对象）

#### 匹配宽度
- **操作**：点击 `宽` 按钮
- **效果**：所有对象的宽度匹配第一个选中对象的宽度
- **注意**：通过调整缩放比例实现，保持纵横比

#### 匹配高度
- **操作**：点击 `高` 按钮
- **效果**：所有对象的高度匹配第一个选中对象

#### 匹配大小
- **操作**：点击 `全` 按钮
- **效果**：同时匹配宽度和高度

---

### 4. 智能吸附（拖动时自动触发）

#### 功能说明
拖动对象时，如果接近其他对象的边缘或中心线，会自动吸附并显示**紫色辅助线**。

#### 吸附规则
- **左边缘对齐**：当前对象左边缘接近其他对象左边缘时
- **右边缘对齐**：右边缘接近其他对象右边缘
- **水平居中**：当前对象水平中心接近其他对象中心
- **顶部对齐**：顶边接近其他对象顶边
- **底部对齐**：底边接近其他对象底边
- **垂直居中**：垂直中心接近其他对象中心

#### 吸附距离
- 默认吸附距离为 **5像素**（视口坐标）
- 最小距离阈值 **0.5像素**（防止已对齐的对象粘在一起）

#### 禁用吸附
- **按住 Shift 键**可以临时禁用智能吸附
- 适用场景：需要精确移动已对齐的对象时

#### 辅助线
- **紫色竖线**：表示水平方向对齐
- **紫色横线**：表示垂直方向对齐
- **自动消失**：松开鼠标后辅助线自动清除

## 🎯 实战示例

### 示例1：制作按钮栏（水平对齐）

```
1. 创建3个矩形（代表按钮）
2. 框选这3个矩形（Ctrl+拖拽）
3. 点击对齐工具栏的 "顶部对齐" ⬆️
4. 点击 "水平分布" ⬌
5. 完成！三个按钮顶部对齐且均匀分布
```

### 示例2：制作网格布局

```
1. 创建9个矩形（3行3列）
2. 先选中第一行3个矩形
3. 点击 "顶部对齐" + "水平分布"
4. 重复操作第2行和第3行
5. 选中第一列3个矩形
6. 点击 "左对齐" + "垂直分布"
7. 重复操作第2列和第3列
8. 完成！完美的3x3网格
```

### 示例3：快速对齐UI元素

```
1. 创建标题文本和多个按钮
2. 选中所有按钮
3. 点击 "匹配高度" 使所有按钮高度一致
4. 点击 "水平分布" 均匀排列
5. 选中标题和按钮组
6. 点击 "水平居中对齐" 使标题和按钮居中对齐
```

## 💡 最佳实践

### 1. 组合使用对齐功能
- 先使用 **对齐** 调整位置
- 再使用 **分布** 调整间距
- 最后使用 **匹配大小** 统一尺寸

### 2. 利用智能吸附
- 拖动对象时观察紫色辅助线
- 紫色线出现时松开鼠标，对象会自动对齐
- 可节省大量手动调整时间
- **按住 Shift 键可以临时禁用吸附**，用于精确移动已对齐的对象

### 3. 搭配多选功能
- 使用 `Ctrl+拖拽` 框选多个对象
- 使用 `Shift+点击` 累加选择
- 对齐工具在选中2个以上对象时最有效

### 4. 批量布局技巧
- 对于大量相似对象，先创建一个
- 使用 `Ctrl+D` 快速复制多个
- 用对齐工具批量调整位置和大小

## ⚙️ 技术细节

### 对齐算法

**左/右/上/下对齐**：
- 计算所有对象的边界框
- 找到最极端的边缘位置
- 将所有对象移动到该位置

**居中对齐**：
- 计算所有对象的中心点
- 求平均值作为目标中心
- 移动对象使其中心对齐到目标

**分布**：
- 按位置排序对象
- 计算总空间和对象总宽度/高度
- 均匀分配剩余空间

**匹配大小**：
- 以第一个选中对象为基准
- 通过缩放比例调整其他对象
- 保持对象的锚点位置不变

### 吸附机制

- 实时计算拖动对象与其他对象的相对位置
- 当距离小于吸附阈值（5px）时触发吸附
- 使用 PixiJS Graphics 绘制临时辅助线
- 鼠标松开时清除所有辅助线

## 🐛 已知限制

1. **容器对象**：容器内的子对象不参与吸附计算
2. **旋转对象**：旋转后的对象对齐可能不够精确（基于边界框）
3. **性能**：对象数量超过100个时，吸附计算可能有轻微延迟
4. **嵌套对象**：父子关系的对象对齐时，只影响直接选中的对象

## 💡 常见问题

**Q: 对齐后对象无法分开怎么办？**  
A: 按住 **Shift 键**再拖动对象，可以临时禁用智能吸附。

**Q: 为什么拖动时对象会自动跳到其他对象位置？**  
A: 这是智能吸附功能。如果不需要，按住 Shift 键拖动即可禁用。

**Q: 吸附的灵敏度可以调整吗？**  
A: 目前吸附距离固定为5像素。未来版本可能会添加设置选项。

---

## 📊 性能优化建议

- 对于大型场景（100+对象），建议关闭智能吸附以提升拖动性能
- 使用分组容器管理相关对象，减少单次对齐操作的对象数量
- 对齐操作支持撤销（Ctrl+Z），可放心实验
